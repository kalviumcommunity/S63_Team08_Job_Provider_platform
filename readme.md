# 📊 SkillSight – Data-Driven Career Guidance System

## 🧠 Problem Statement

Job seekers often struggle to understand which skills are currently in demand and which ones lead to better placement opportunities. Due to lack of data-driven insights, students may invest time in learning irrelevant or low-impact skills, resulting in poor career outcomes.

---

## 🎯 Objective

The goal of this project is to analyze labour market data and extract meaningful insights that help students:

* Identify **trending and in-demand skills**
* Understand **salary distribution across skills**
* Discover **relationships between skills and job opportunities**
* Make **informed career decisions**

---

## 💡 Solution Approach

This project uses data science techniques to process and analyze job market datasets. By performing exploratory data analysis (EDA) and visualizing patterns, we transform raw data into actionable insights for career planning.

---

## 📂 Dataset

The dataset contains information such as:

* Job Titles
* Required Skills
* Salary Information
* Experience Levels
* Location

(Source: Public job datasets such as Kaggle / LinkedIn job postings)

---

## ⚙️ Tech Stack

* **Programming Language:** Python
* **Libraries Used:**

  * Pandas – Data manipulation and analysis
  * NumPy – Numerical operations
  * Matplotlib & Seaborn – Data visualization
* **Environment:** Jupyter Notebook

---

## 🔄 Project Workflow

### 1. Data Collection

* Import dataset from CSV/JSON files

### 2. Data Cleaning

* Handle missing values
* Remove duplicates
* Standardize skill names

### 3. Exploratory Data Analysis (EDA)

* Analyze most demanded skills
* Study salary distribution
* Compare skills across job roles
* Identify trends in job market

### 4. Data Visualization

* Bar charts for top skills
* Line graphs for trends
* Heatmaps for correlations
* Pie charts for distribution

### 5. Insight Generation

* Extract meaningful conclusions from data
* Highlight key skills and career paths

---

## 📈 Key Features

* 📊 Analysis of **top trending skills**
* 💰 **Salary vs skill comparison**
* 📉 Demand trends across job roles
* 📍 Insights based on experience levels and locations

---

## 🧪 Sample Insights

* Skills like Python, SQL, and Data Analysis show high demand
* AI/ML-related skills are rapidly growing
